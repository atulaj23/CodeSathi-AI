# ⚡ CodeSathi AI

**Build Smarter. Ship Faster.**

CodeSathi AI is an intelligent AI-powered workspace built for developers, students, and healthcare professionals — combining AI code generation, debugging assistance, and healthcare information tools in one unified platform.

🔗 **Live Demo:** [code-sathi-ai.vercel.app](https://code-sathi-ai.vercel.app/)

---

## 📌 Overview

CodeSathi AI brings together two powerful AI workspaces under one platform:

- **Developer Workspace** — Generate code, debug projects, and get instant help across multiple programming languages.
- **Healthcare Workspace** — Understand symptoms, medical reports, and medicine information through an AI assistant designed to educate, not replace medical professionals.

---

## ✨ Features

### 🧑‍💻 AI Developer Tools
- **AI Code Generation** — Generate complete applications in React, Node.js, Python, Java, C++, PHP, and more.
- **Instant Debugging** — Find bugs, improve performance, and optimize code automatically.
- **AI Memory** — Continue previous chats and work on large projects without losing context.

### 🏥 Healthcare AI Tools
- **Symptom Checker** — Understand possible causes based on symptoms and receive educational guidance.
- **Report Analysis** — Upload lab reports and understand medical terms in simple language.
- **Medicine Information** — Learn about medicines, common uses, and precautions.

> ⚠️ **Medical Disclaimer:** Healthcare Sathi AI provides educational information only. It cannot diagnose diseases, prescribe medicines, or replace qualified doctors. In case of a medical emergency, contact local emergency services immediately.

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- Python 3.x installed
- An API key for the AI provider used in this project (e.g. OpenAI/Gemini/etc.)

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/codesathi-ai.git
cd codesathi-ai

# Install frontend dependencies
npm install

# Install backend dependencies
pip install -r requirements.txt
```

### Environment Variables

Create a `.env` file in the root directory and add:

```
API_KEY=your_api_key_here
```

### Run Locally

```bash
# Start frontend
npm run dev

# Start backend (in a separate terminal)
python app.py
```
